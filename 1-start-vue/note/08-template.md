# 템플릿 문법 - 기본

- 템플릿 문법?
  - 뷰로 화면을 조작하는 방법
  - 데이터 바인딩과 디렉티브가 있음

## 데이터 바인딩

- Mustache Syntax `{{}}`
- 실습: [data-binding.html](../playground/data-binding.html)

## 디렉티브

- `v-` 로 시작하는 뷰전용 요소의 속성
- `v-if` `v-show` 의 차이
  - v-if 는 DOM 을 갱신하고
  - v-show는 돔에 남아있지만 css `display: none;` 으로 가려짐
- 실습: [methods.html](../playground/methods.html)
